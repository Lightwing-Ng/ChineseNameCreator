## Descriptions
<p>Feel puzzle about having a Chinese Name, try this.</p>
## How to run:
<p>Shift the current diretory to ../ChineseName.py</p>
```
cd ../ChineseName/ChineseName
```
Then
```
python ChineseName.py
```
